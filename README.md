# Mahjong Tracker

A mobile-first mahjong score tracking web app built with Next.js.

## Features
- 建立牌局 / 房間
- 即時記錄分數變化
- 支援 undo / 歷史紀錄
- 分數趨勢圖
- 中文化手機優先 UI

## Tech Stack
- Next.js
- React
- TypeScript
- Tailwind CSS
- Recharts
- Supabase
